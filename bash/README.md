# Bash / Shell Support

Language support for Bash and shell scripts — diagnostics, completions, hover documentation.

**LSP**: `bash-language-server`
**File types**: `.sh`, `.bash`, `.zsh`, `.fish`

## Prerequisites

- **Node.js** and npm must be installed.

The LSP server is installed automatically when you install this extension. If you need to install it manually:

```
npm install -g bash-language-server
```

## Features

- Syntax errors and shellcheck diagnostics
- Completion for commands, variables, and built-ins
- Hover documentation for commands
- Go-to-definition for functions and variables
