# XML Language Support

Language support for XML — formatting, validation, and schema support.

**LSP**: `lemminx` (Eclipse LemMinX)
**File types**: `.xml`, `.xsd`, `.xsl`, `.xslt`, `.svg`, `.pom`

## Prerequisites

- **lemminx** must be installed manually and placed on PATH. Download from the Eclipse LemMinX releases page.

Note: lemminx cannot be auto-installed — download the binary for your platform.

## Features

- Syntax validation and well-formedness checking
- XSD/DTD schema validation and completion
- Formatting (`:Lformat`)
- Tag auto-completion
